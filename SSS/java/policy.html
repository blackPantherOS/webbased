Setting the Java Security Policy for webCDcreator

  1. Save the certificate jhaeger.x509, the policy java.policy, and the Java application Policy.class to the same directory. (If you have the JDK 1.2 or above installed, you may compile Policy.java yourself by "javac Policy.java".)
  2. Open a command window (like MS-DOS Prompt, Command Prompt, xterm, ...).
  3. Change to the directory, you copied the above files to.
  4. Execute 
    java Policy
    and follow the instructions.
  5. Restart your browser.
